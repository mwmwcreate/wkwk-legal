# プライバシーポリシー

最終更新日: 2026年4月22日

ina（以下「本アプリ」）をご利用いただきありがとうございます。本プライバシーポリシーは、本アプリが取得・利用する情報について説明します。

## 1. 事業者情報

- サービス名: ina
- 運営者: mwmw
- 連絡先: mwmw.create@gmail.com

## 2. 取得する情報

本アプリは、利用者のメールアドレス・電話番号・氏名・住所など個人を識別できる情報は一切取得しません。以下の情報のみを取得・保存します。

### 2-1. 利用者がアプリに入力する情報

- ニックネーム（任意の文字列）
- プロフィール画像（任意、端末内および Firebase Storage に保存）
- 投稿の内容（タイトル・本文・画像・タグ）
- 「イーナ！」リアクション履歴
- 通報内容、ブロックしたユーザーID

### 2-2. 自動的に生成・取得する情報

- 匿名利用者ID（端末内で生成される UUID。個人とは紐づきません）
- プッシュ通知用トークン（FCMトークン）
- 投稿の作成日時

本アプリは広告 SDK・分析 SDK を使用しておらず、IDFA（広告識別子）・位置情報・連絡先・健康情報・購入履歴などは取得しません。

## 3. 利用目的

取得した情報は以下の目的にのみ利用します。

- 投稿を他の利用者のタイムラインに表示する
- AI による投稿分類・価値観プロファイル生成の提供
- 「イーナ！」リアクションがついたことの通知
- 通報・ブロック機能による安全性の確保
- 不具合調査、サービスの改善

## 4. 第三者提供

以下の外部サービスを利用しており、機能提供のために必要最小限のデータが送信されます。

| 送信先 | 送信データ | 目的 |
|---|---|---|
| Google Firebase (Firestore / Storage / Cloud Messaging / Cloud Functions) | 投稿内容、画像、匿名ID、FCMトークン | データ保存・通知配信 |
| OpenAI (API経由) | 投稿のタイトル・本文、ユーザーが入力したチャット文 | AI による分類・分析 |
| Apple Push Notification service | FCMトークン | プッシュ通知配信 |

OpenAI への送信内容は OpenAI 社のプライバシーポリシーおよび API 利用規約に従って取り扱われます。

上記以外の第三者に対して、個別に識別可能な形で利用者のデータを販売・共有することはありません。

## 5. データの保存と削除

- 投稿データは Firebase (Google Cloud) 上に保存されます。
- 利用者はアプリ内の「設定」→「アカウントを削除」から、いつでも自身の投稿・イーナ！履歴・ブロックリスト等を削除できます。
- アカウント削除後、サーバー上のデータは速やかに削除されます（バックアップ上は最大30日間残存する場合があります）。

## 6. 年齢制限

本アプリは 13 歳以上を対象としています。13 歳未満の方は保護者の同意なしに本アプリを利用しないでください。

## 7. 不適切なコンテンツへの対応

本アプリはユーザー生成コンテンツ（UGC）を扱うため、以下の方針で運営します。

- 投稿は通報・ブロック機能により利用者自身が管理できます。
- 通報を受けた投稿は原則として 24 時間以内に確認し、規約違反が認められた場合は削除します。
- 嫌がらせ・性的・違法・暴力的など不適切な投稿は一切許容しません。

## 8. セキュリティ

- サーバーとの通信は HTTPS により暗号化されています。
- OpenAI API キーはサーバー側（Firebase Functions）でのみ保管し、アプリには埋め込んでいません。

## 9. ポリシーの変更

本ポリシーは必要に応じて改定することがあります。重大な変更がある場合はアプリ内または本ページで告知します。
